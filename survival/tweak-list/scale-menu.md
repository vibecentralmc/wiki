# Scale Menu

If players VIP tier 5 and above desire they can change their characters size in the `/scale` menu.\

Available size options are:
* 0.4x (72cm / 2'4")
* 0.5x (90cm / 2'11")
* 0.6x (108cm / 3'6")
* 0.7x (126cm / 4'2")
* 0.8x (144cm / 4'9")
* 0.9x (162cm / 5'4)
* 1.1x (198cm / 6'6")
* 1.2x (216cm / 7'1")
* 1.3x (234cm / 7'8")
* Return to Normal size (1.0x)
